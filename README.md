## Project Goals & Scope

**Problem:** Modern web development projects often lack structured AI-assisted development practices, leading to inefficient workflows and poor documentation.

**Solution:** An AI-assisted capstone project demonstrating modern development workflows with proper tooling, version control, and AI-powered code assistance.

**Objectives:**
- Build a web application using Node.js and Express
- Demonstrate AI-assisted development workflows with documented prompts and iterations
- Deliver a deployable MVP with user authentication, data management, and responsive UI

**Out of scope (for v1):** Mobile app, payment integration, advanced AI model training
